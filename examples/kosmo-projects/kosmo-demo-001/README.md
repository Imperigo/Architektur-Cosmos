# Kosmo Demo 001

Minimal local project package for Kosmo MVP 0.1.

Purpose:

- test the shared package structure
- give Kosmo Design a simple model profile to read/write
- give Kosmo Draw and Kosmo Viz predictable output locations
- give Kosmo Publish a review pack to assemble
- give Kosmo Zentrale logs and gates to track

This demo is intentionally conceptual. It should not be treated as a real competition submission or a public-safe project.
