# Kosmo Brief

Project: Kosmo Demo 001  
Status: local review only

## Intent

Create a small conceptual architecture workspace that proves the Kosmo MVP 0.1 flow:

- read a brief
- structure constraints
- create a first room and story model
- export simple plans
- prepare a review package

## Program

- shared studio / work room
- project library / reference wall
- small meeting room
- model / material bench
- service core

## Design Direction

The demo should feel like a compact architecture lab rather than a finished building. The first geometry should be simple, legible and easy to regenerate from data.

## Success Criteria

- Kosmo Design can read `design/model-profile.json`.
- Kosmo Draw can replace the placeholder SVGs with generated plans.
- Kosmo Publish can assemble the review pack.
- Kosmo Zentrale can register decisions and uncertainties.
