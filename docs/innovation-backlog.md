# Innovation Backlog

These ideas are intentionally not part of the current implementation. They define the next frontier once the semantic zoom atlas is stable.

## Curated Study Paths

Guided routes through the atlas, for example Modernism, Housing, Reuse, Landscape, Theory, or Urban Infrastructure. A path should behave like a highlighted sequence in the atlas, not like a separate article page.

## Source Trail

Each entry can later expose sources, lecture references, books, image credits, drawing origins, and external archives. The source trail should attach to the project dossier level.

## Comparison Mode

Two entries can be pinned and compared by year, typology, media, themes, relations, and position in the atlas.

## Typology Lens

A lens that emphasizes recurring architectural types across time: villa, temple, housing block, pavilion, garden, museum, infrastructure, campus, plan, and manifesto.

## Image Deep Zoom

Plans, sections, and high-resolution archival images can later use tile-based viewing. OpenSeadragon is the likely candidate once real image assets exist.

## 3D Model Slot

The media model should later accept a fifth representation: a generated or uploaded 3D model. This is explicitly deferred until the four canonical project media slots work well.
