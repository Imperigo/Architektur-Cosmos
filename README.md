# Architektur-Cosmos-Browser

Working title / concept name: **Architecture Universe**

A radial zoomable architecture archive and reference atlas. The project organizes architectural history as a circular knowledge field with concentric time rings, radial style sectors and expandable entries.

## MVP

The first version builds a React/SVG-based radial atlas without filters, search, database or CMS. It tests the core interaction: zoom, pan, project nodes and a detail panel.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Next.js URL shown in the terminal.
