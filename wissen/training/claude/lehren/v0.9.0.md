# Lehren aus v0.9.0 «Heimkraft» (Release 22.07.2026)

## Gates
- **Client-seitige Blockade sieht aus wie Netzausfall.** Chips orange, curl daneben grün
  → IMMER zuerst die eigene CSP (`connect-src`) prüfen, bevor Netz/Firewall debuggt wird.
  Der Tailnet-Blocker sass in index.html UND tauri.conf.json (Beleg: 198f01d, ROADMAP 606);
  die Diagnose-Heuristik (`istWahrscheinlichCspGeblockt`) muss bei jeder CSP-Änderung mitziehen.
- **Vertrags-Enum-Symmetrie:** Schreibt ein Worker einen neuen ehrlichen Status, MUSS er in den
  zod-Vertrag — sonst lehnt `safeParse` die Antwort ab und der Poll verschluckt genau die
  Ehrlichkeit, die man bauen wollte (`kein-render-worker` fehlte in RenderJobStatus; Blender
  hatte das Muster richtig. Beleg: e828058, ROADMAP 610, Matrix-C-1-Prüfer).
- **Adversariale Matrix lohnt sich auch bei grünen Suiten:** C-1 fand einen echten Vertragsbruch,
  C-5 einen sachlichen Fehlbeleg im Konzeptdokument — beides für Suiten unsichtbar.
- **Band-Beweise im sichtbaren Fenster messen:** Der Zoom-Schrift-Beweis (E-K27a) muss an der
  LOD-Lesegrenze (scale ~0.02, «mittel») gemessen werden, nicht an der 0.005-Klemme — dort
  blendet planLod die Dims per display:none aus (zoom-text.spec.ts, zoomeAnLesegrenze).

## Konventionen
- **Nach jedem gegateten Teilpaket sofort committen+pushen.** Der Container-Rollback am 22.07.
  frass das komplette ungecommittete Owner-Punkte-Paket (Cursor+Fehlermeldeweg) — es musste aus
  dem Kontext rekonstruiert werden (608/609 sind die zweite Fassung); ein zweiter Restart am
  selben Tag bestätigte die Regel. origin ist die Wahrheit.
- **cwd-Falle bleibt tödlich:** `git add` mit Root-Pfaden aus einem Unterverzeichnis = Exit 128;
  `npm run typecheck` aus tools/ = vakuos-grün. Git/ROADMAP/Golden-Checks NUR vom Repo-Root,
  npm/vitest/playwright NUR aus kosmo-orbit.
- **test_*.py ohne pytest-Kompatibilität ist eine Falle:** `sys.exit()` auf Modulebene lässt
  `python3 -m pytest` mit INTERNALERROR sterben. Muster: `if __name__ == "__main__"`-Guard +
  ein Sammel-`def test_…()` (f00499d/610).
- **Custom Property als Cursor-Spiegel:** globales `cursor:none !important` tötet das
  computed-cursor-Signal; `--k-zeiger` erbt wie `cursor` selbst und trägt die Zonen-Form
  (ccc5597, 14 Deklarationen + Viewport3D-Inline-Spiegel).

  CSP-Fix war das der zweite Riegel; Fix ist `OLLAMA_ORIGINS=*` in der systemd-Unit

- **iPad-/Remote-Zustellung:** Web-Edition vom Server erhält Fixes via Auto-Update sofort,
  Desktop-Installer erst mit dem Release-Build — bei client-seitigen Fixes IMMER beide Wege
  benennen (Browser-Workaround kommunizieren, Installer nachliefern).
